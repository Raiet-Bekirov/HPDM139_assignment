Minutes – Intersectional Fairness Toolkit Project (Final Meeting)¶

Date: 5 January 2026 Time: 8:00 PM Format: Online

1. Attendance¶

All group members present:

  • Nick Berry
  • Kayla Yasmine
  • Becky Griffiths
  • Raiet Bekirov

2. Purpose of the Meeting¶

The purpose of the meeting was to conduct a final review of the project prior to submission, including:

  • Review of the codebase, repository structure, and Python package
  • Confirmation that the package installs and runs correctly
  • Identification of any final changes or fixes required
  • Agreement on readiness for submission and final responsibilities

3. Review of Codebase and Package¶

The group reviewed the final state of the repository and confirmed that:

  • Core fairness metrics and intersectional metrics are implemented and standardised
  • Visualisation functions are working as intended
  • Example notebooks run end-to-end without errors
  • Nick confirmed that the package installs correctly in a clean environment, indicating that dependencies and packaging are correctly specified.

Decision: The group agreed that the codebase is complete and functional.

4. Documentation and Demonstration Environment¶

Documentation (tutorial, API reference, and design decisions) was reviewed and judged to be clear and suitable for assessment.

The Binder environment was discussed to ensure it points to the final released package version rather than a development snapshot.

5. Actions Agreed¶

The following final actions were agreed to complete submission preparation:

  • Kayla: Lint Jupyter notebooks and add final references
  • Becky: Update Binder configuration to point to the final PyPI release
  • Nick: Upload version v1.0.0 of the package to PyPI
  • Nick: Submit the project on behalf of the group, send submission confirmation email to all group members

6. Submission Readiness¶

The group agreed that the project meets the requirements of the HPDM139 group assignment.

No further meetings were deemed necessary.

Decision: The project was approved for final submission.
