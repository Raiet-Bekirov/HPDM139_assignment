Minutes – Intersectional Fairness Toolkit Project Meeting 6

Date: 3/1/2026 Format: Online

1. Attendance

Becky, Kayla, Nick, Raiet

2. Metrics Review

The group reviewed the current fairness metric implementations, including group-based and intersectional measures.

It was agreed that the existing metric functions are sufficient for the scope of the project, provided they are merged and standardised.

Decision:

Kayla will merge her metrics code with Becky’s existing functions to create a unified metrics module.

3. Visualisations Review

Current visualisation functions were discussed.

The group agreed that visualisations add interpretability and strengthen the demonstration of fairness concepts.

Decision:

Raiet will test the visualisation functions and extend the example notebook to include fairness visualisations.

4. Documentation and Demo Environment

The importance of clear documentation for assessment and usability was emphasised.

A Binder-based environment was discussed to allow easy execution of demo notebooks.

Decisions:

Becky will activate and test a Binder environment for the demonstration notebooks.

Nick will produce and finalise documentation, including:

Tutorial

API reference

Design decisions

5. Integration Plan

The group agreed that all components are now close to completion.

The final phase will focus on integration, testing, and documentation polish.

6. Actions Agreed

Member Action Kayla Merge metrics with Becky’s functions Raiet Test visualisations and extend example notebook Becky Set up Binder environment for demo notebooks Nick Produce and finalise documentation (tutorial, API, design decisions)
